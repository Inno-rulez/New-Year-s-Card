# Happy New Year Card

This repository contains a small, single-page "Happy New Year" card you can open locally and share with loved ones.

Files added:

- `index.html` — the card markup
- `styles.css` — styling and responsive layout
- `script.js` — confetti rain and simple controls

To preview locally open `index.html` in your browser (double-click or right-click -> Open With).

Quick deploy options:

- GitHub Pages: create a repo, push these files to the default branch, and enable Pages in repo settings.
- Netlify: drag-and-drop the folder into Netlify Drop or connect your git repo.

Notes:

- The confetti is implemented with lightweight DOM elements and CSS animations.
- To save the card as an image, use your browser's Print -> Save as PDF or take a screenshot.

Enjoy — and Happy New Year from Mdu!
